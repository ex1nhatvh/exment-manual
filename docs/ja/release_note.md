# リリースノート

## v1.0.12(2019/04/11)
- バックアップ & リストアで、成功しているにも関わらずエラーアイコンが表示される問題の修正

## v1.0.11(2019/04/11)
- 全データ編集権限をもたないユーザーがデータ編集時、権限登録項目を表示しないよう修正
- URLエンドポイントの作成方法を見直し、パフォーマンス改善
- 役割を削除できない不具合を修正
- データのフィルターに、「作成ユーザー」「更新ユーザー」を指定していると、エラーになる不具合の修正
- 最新版をインストールしていても、TOPページに「最新バージョンがあります」と表示される不具合を修正
- ユーザー設定画面で、パスワード変更画面が表示されない不具合の修正
- ログインユーザー画面で、パスワード自動生成にするとメールが送信されない不具合の修正
- ログインユーザー画面で、メール設定の不備によりメールが送信されないと、メッセージを表示するよう変更
- バックアップ画面のアイコンなど、一覧画面内の一部の項目へオンマウス時にツールチップを表示。v1.1.0対応時に完全対応されます。
- APIをプレサポートしました。仕様はv1.1.0公開時に掲載します。
- 一部のURLを、相対パスから絶対パスに修正。v1.1.0対応時に完全対応されます。
- その他、軽微な修正

## v1.0.10(2019/03/27)
- ドキュメント出力にPDF形式を追加(近日、マニュアルを修正いたします)
- PHP7.3で動作していなかった問題の修正
- カスタム列の検索インデックスを、テーブルごとに10件まで登録できるようルール追加
- カスタム列の登録画面で、オプションの必須チェックを追加
- カスタム列の列種類「画像」で、画像のみ選択できるよう修正
- カスタムフォームの「データ連動設定」が正常に登録できなかった問題の修正
- データ削除時に、テーブルの選択肢として登録しているデータがあれば、削除できないよう変更
- 初期設定画面で、スクリプトエラーが発生していた問題の修正
- Exmentの最新バージョン情報の取得に失敗していたときに、システムエラーが発生していた問題の修正
- 外部サーバーとの接続を行わないモードの追加
- "exment:install"ならびに"exment:update"コマンドで、public、lang、viewsフォルダが正常に配布されない問題の修正
- 未使用のクラスの削除（実装の検証で使用していたクラスを含んでおりました。大変申し訳ございません）
- その他、リファクタリング等

## v1.0.9(2019/03/14)
- exment:updateコマンドを追加。更新時に実施する必要のあるコマンドを削減しました
- Exmentインストール時に必要なコマンドを削減
- 権限設定で、複数の権限を設定していた場合に発生する不具合を修正
- 権限設定の内部処理を修正（クラス名の変更など）
- ページ内に配置しているマニュアルへのリンクを追加
- パッケージから、マニュアルのページや画像を削除。マニュアル関係は別パッケージに移行しました
- その他、軽微な修正

- データコピー設定の削除ができない不具合の修正

- その他、軽微な修正
