Please check this repository if you want to interest Exment.

# How to debug manual
this manual uses [docsify](https://docsify.js.org/#/)

- Install node.js

- Install docsify-cli

~~~
npm i docsify-cli -g
~~~

- Clone this repository

- Execute this command on root directory

~~~
docsify serve docs
~~~

- Show http://localhost:3000 on browser.
